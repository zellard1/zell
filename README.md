# ARIA: La couronne, le sceptre et l'orbe

Ce dépôt contient un petit jeu narratif jouable dans le terminal. Il
s'inspire librement de l'univers du livre *ARIA : La couronne, le
sceptre et l'orbe* et du style sombre de *Slay the Princess*.

Le jeu est volontairement minimaliste : tout est en noir et blanc avec
de simples illustrations ASCII et sans doublage audio (celui-ci peut
être ajouté ultérieurement en modifiant la fonction `play_line`).

## Lancer le jeu

Exécutez simplement :

```bash
python3 aria_game.py
```

Puis suivez les instructions à l'écran pour choisir vos actions et
explorer les différents embranchements narratifs.
