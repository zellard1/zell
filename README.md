# Chef DEGAS - Site Web de Recettes de Pâtisserie

## Présentation

Chef DEGAS est un site web statique dédié aux recettes de pâtisserie créées par Claude (Opus 4). Le site permet aux utilisateurs de découvrir des recettes de viennoiseries, entremets et tartes, avec des fonctionnalités interactives comme l'ajustement des portions, la gestion des favoris et le partage sur les réseaux sociaux.

## Structure du projet

```
chef-degas/
├── index.html                  # Page d'accueil
├── compte.html                 # Page de compte utilisateur
├── a-propos.html               # Page À propos
├── mentions-legales.html       # Mentions légales
├── 404.html                    # Page d'erreur 404
├── viennoiseries/              # Catégorie Viennoiseries
│   ├── index.html              # Liste des viennoiseries
│   └── croissants.html         # Exemple de recette
├── entremets/                  # Catégorie Entremets
│   └── index.html              # Liste des entremets
├── tartes/                     # Catégorie Tartes
│   └── index.html              # Liste des tartes
├── assets/                     # Ressources
│   ├── css/                    # Feuilles de style
│   ├── js/                     # Scripts JavaScript
│   └── images/                 # Images du site
├── documentation_technique.md  # Documentation technique complète
├── guide_ajout_recettes.md     # Guide pour ajouter de nouvelles recettes
└── guide_maintenance.md        # Guide de maintenance du site
```

## Fonctionnalités

- **Navigation responsive** adaptée à tous les appareils
- **Présentation des recettes** avec informations clés, ingrédients et étapes
- **Ajustement des portions** avec recalcul automatique des quantités
- **Système de favoris** utilisant le stockage local du navigateur
- **Partage sur réseaux sociaux** pour chaque recette
- **Interface de connexion/inscription** (simulation)
- **Design accessible** respectant les normes WCAG/RGAA

## Technologies utilisées

- HTML5
- CSS3 (avec design responsive)
- JavaScript (vanilla)
- Font Awesome pour les icônes
- Google Fonts pour la typographie

## Installation

Le site étant entièrement statique, il suffit de copier tous les fichiers sur un serveur web ou un hébergement statique. Aucune configuration serveur particulière n'est nécessaire.

## Documentation

Trois documents sont fournis pour faciliter la gestion du site :

1. **Documentation technique** (`documentation_technique.md`) : Description détaillée de l'architecture, des fonctionnalités et des technologies utilisées.

2. **Guide d'ajout de recettes** (`guide_ajout_recettes.md`) : Instructions pas à pas pour ajouter de nouvelles recettes au site.

3. **Guide de maintenance** (`guide_maintenance.md`) : Procédures pour la maintenance régulière, l'optimisation et la résolution des problèmes courants.
